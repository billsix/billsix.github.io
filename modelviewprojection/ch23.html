Towards Standard Projection Matrix Pt 2 Demo 23¶

Purpose¶

Remove cameraSpace.z entirely from the perspective matrix.

This will allow us to use one perspective projection matrix for all vertices, independent of the z coordinate of each input vertex.

How to Execute¶

On Linux or on MacOS, in a shell, type “python src/demo24/demo.py”. On Windows, in a command prompt, type “python src\demo24\demo.py”.
